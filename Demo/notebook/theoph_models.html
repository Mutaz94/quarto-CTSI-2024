General info on YAML:

  • The YAML Header is marked by the three dashes “- - -”

  • The general principle is to have: the name of a data item (a key), followed by a colon, a space, and then the data item’s value. A key-value pair in the format key: value.

  • Each line in YAML is a new item.

  • Dashes (-) represent individual items in a list.

  • Note that indentations matter in YAML!!

  • YAML can be used to specify the global settings for your document (i.e. figure caption location, citation method, code folding, etc.).

  • You can use the tab key to see what options are available

1.1 Introduction

Theophylline is a medication used to treat asthma and chronic obstructive pulmonary disease as a second-line drug. It is a bronchodilator. This activity reviews the indications, action, and contraindications for theophylline as a potential agent in treating asthma and chronic obstructive pulmonary disease. This activity will highlight the mechanism of action, adverse event profile, pharmacokinetics, and drug interactions pertinent for members of the interprofessional team in the treatment of patients with asthma and chronic obstructive pulmonary disease.
